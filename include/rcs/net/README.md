# net

网络接入层头文件：连接管理、协议编解码、心跳与限流相关接口。

## 当前职责

- `message.hpp`：定义网络消息、长度前缀帧头，以及基础编解码工具。
- `rate_limiter.hpp`：提供单连接消息频率限制。
- `tcp_connection.hpp`：封装一个 TCP 客户端连接的异步读写、发送队列和生命周期。
- `network_gateway.hpp`：封装 TCP 监听入口、连接管理、消息回调、单播和广播。

## 设计边界

网络接入层只负责连接、帧协议、心跳、断线和限流。登录鉴权、玩家会话、房间匹配等业务逻辑通过 `NetworkGateway` 的回调接入，避免网络层直接依赖上层模块。
